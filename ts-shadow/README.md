# TypeScript Shadow Generator Web App

A web-based version of the realistic shadow generator using TypeScript and HTML5 Canvas.

## Features

- Upload foreground and background images
- Adjust light angle and elevation
- Generate composite with realistic shadow
- View debug outputs: shadow only and mask
- **Bonus**: Depth map warping (shadow bends based on depth)

## Usage

1. Run `npm run dev`
2. Open the browser at the provided URL
3. Upload images
4. Adjust parameters
5. Click Generate

## Implementation

Uses Canvas 2D API for image processing and compositing.
